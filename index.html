Options


Option Type Description
format* String | Array The date format, can be a string or an array of strings.
It is MM/DD/YYYY, by default
maxDate String | Date The max date.
minDate String | Date The min date.
separator* String Used to separate the day, month, and year.
Most popular examples are a slash (/), hyphen (-), or dot (.)
It is /, by default.

* — Required options if different from default values.

The following table describes the token meanings:

Token Description
D Day of the month as digits; no leading zero for single-digit days.
DD Day of the month as digits; leading zero for single-digit days.
M Month as digits; no leading zero for single-digit months.
MM Month as digits; leading zero for single-digit months.
YY Year as last two digits; leading zero for years less than 10.
YYYY Year represented by four digits.
h Hours; no leading zero for single-digit hours (12-hour clock).
hh Hours; leading zero for single-digit hours (12-hour clock).
H Hours; no leading zero for single-digit hours (24-hour clock).
HH Hours; leading zero for single-digit hours (24-hour clock).
m Minutes; no leading zero for single-digit minutes.
mm Minutes; leading zero for single-digit minutes.
s Seconds; no leading zero for single-digit seconds.
ss Seconds; leading zero for single-digit seconds.
A* AM PM
a* am pm

* — A and a used only with hh or h.

Below are some examples of possible formats:

  • YYYY/DD/MM
  • YYYY-MM-DD
  • YYYY.MM.DD
  • DD/MM/YYYY
  • DD-MM-YYYY
  • DD.MM.YYYY
  • MM/DD/YYYY
  • MM-DD-YYYY
  • MM.DD.YYYY
  • M/D/YY
  • M-D-YY
  • M.D.YY
  • YYYY-MM-DD HH:mm:ss
  • YYYY-MM-DD H:m:s
  • YYYY-MM-DD hh:mm:ss A
  • YYYY-MM-DD hh:mm:ss a
  • YYYY-MM-DD h:m:s A
  • YYYY-MM-DD h:m:s a
  • ...
